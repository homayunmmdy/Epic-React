# Hello word in JS

```javascript
// Create outer div with ID 'root' and append to body
const root = document.createElement("div");
root.id = "root";
document.body.appendChild(root);

// Create inner div with class 'container' and text 'Hello World'
const container = document.createElement("div");
container.classList.add("container");
container.textContent = "Hello World";

// Append inner div to the outer div
root.appendChild(container);
```

# Hello World in Raw React

```javascript
import React from 'react';
import ReactDOM from 'react-dom';

// Create React elements
const container = React.createElement(
  'div',
  { className: 'container' },
  'Hello World'
);

const rootElement = React.createElement(
  'div',
  { id: 'root' },
  container
);

// Render to the DOM
ReactDOM.render(rootElement, document.body);
```
